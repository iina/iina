Contribute

IINA is an open source project that welcomes everyone's contributing.

You can contribute to IINA as a user by:

  • Download and test it :P
  • Draft a bug report
  • Send a feature request
  • Localize website and application

If you are a developer, you may…

  • Star, fork or raise issues at github
  • Give me suggestions on coding (I know my code sucks)
  • Join me and develop together

Any feeback is appreciated! Let's make IINA the best media player on macOS.
